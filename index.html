HTML&CSS CheatSheet

The following table was made to be a helpfull instrument for beginners.

HTML Table Reference


Table Tags


Tag Name Description
<table> Table The wrapper element for all HTML tables.
<thead> Table head The set of rows defining the column headers in a table.
<tbody> Table body The set of rows containing actual table data.
<tr> Table row The table row container.
<td> Table data The data in table row.
<tfoot> Table foot The set of rows defining the footer in a table.

Table Attributes


Attribute Name Description
colspan Column span Defines how many columns a td element should span.
rowspan Row span Defines how many rows a td element should span.

HTML Forms


HTML Forms elements


Element Name Purpose
<form> form Form is used to collect and send info to an external source.
<input> input Input is used to render a veriety of input fields.
<label> label Label provides identification for specific input.
<select> select Select can be used to create a dropdown list.
<datalist> datalist Datalist provides an input with the specified list of options.
<textarea> textarea Textarea is used when creating a text-box for multi-input.

HTML Input types


Type Name Description
"checkbox" Checkbox Is used when user can choose seceral options from the list.
"text" text Renders a single row field that uaer can type text inside

CSS selectors


Selector Example Description
tag h1 Refers to all html-elements with selected tag
.cursive class Refers to all html-elements with selected value of class.
#about-me id Refers to a single element with the selected id.
